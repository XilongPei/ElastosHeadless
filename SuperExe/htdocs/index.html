Elastos Headless v5.0

Elastos is an operating system that is designed with universal apps in mind. It facilitates applications developed in any of three kinds of languages, i.e., C/C++, Java and HTML/JS, to interact with each other automatically. Elastos is Android compatible and open source.

Release Notes

Elastos Headless v5.0

This version is a major improvement since the previous series: improved TLS, add additional support for OSX and BSD, the scheduler is aware about protocol handlers (HTTP2 is coming!), a new HTTP parser have been implemented, the plugins can be static or dynamic, the build system is now based on CMake, general performance have been improved and much more. We invite you to read our official release notes:

http://elastos.org

Design, goals and Vision

Performance & Innovation

The goal of Elastos OS is different from Android in at least four aspects:

1. Elastos has a complete set of C/C++ APIs, so it's a better fit for embedded systems and systems with detached peripherals.

2. Elastos is a distributed OS across the Internet to improve end-to-end security and integrity of user experiences.

3. Elastos is pioneering a service oriented (componentized) architecture that is designed to execute in a container/virtual machine. Using this technology, Elastos can take advantage of universal applications that can run on phones, PCs, game consoles, servers, and cloud centers, etc., regardless of CPU types, screen sizes, I/O scenarios (e.g. keyboard, mouse, touch, remote control, etc.).

4. Elastos is decentralized build on top of blockchains and p2p networks, so nobody controls the Elastos eco system, i.e., you may feel free to implement your own markets, search engines, location based services, advertisement agents, and so on.

Ultimately, our passion is to build a new innovative Internet infrastructure that is useful to the world.
